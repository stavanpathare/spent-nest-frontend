# 🏡 Spent Nest

**Spent Nest** is a modern personal finance and expense management platform designed to help users track expenses, manage budgets, analyze spending habits, and gain better control over their financial life.

## ✨ Features

### 💰 Expense Management

* Add, edit, and delete expenses
* Categorize transactions
* Track spending history
* Filter expenses by date, category, and amount

### 📊 Budget Planning

* Create monthly budgets
* Set category-wise spending limits
* Monitor budget utilization
* Receive overspending alerts

### 📈 Financial Analytics

* Interactive dashboards
* Expense trend visualization
* Category-wise spending breakdown
* Monthly and yearly reports

### 🎯 Savings Tracking

* Set savings goals
* Monitor goal progress
* Track financial milestones

### 🔐 Secure Authentication

* User registration and login
* JWT-based authentication
* Protected routes and APIs
* Secure password encryption

### 📱 Responsive Design

* Mobile-friendly interface
* Optimized for tablets and desktops
* Clean and intuitive user experience
